I've just sat down to start on the clues for all the boxes we put out. If
you end up going to Kaikoura before I've "officially" done the clues, send
me an email and I'll send an advanced copy of the clues. Kaikoura is a
fabulous place for boxing because it is so rocky - there is definitely more
scope for putting out boxes in the "shag" area so go prepared!! Other spots
in NZ can be much more tricky. The tops of hills (which Wellington excels
at) tend to be pretty bare, the ground tends to be clay rather than rocks,
and the trees tend to be pines which don't have as many nooks and crannies.
We also had more trouble finding spots in the Blenheim region - but we
persevered and got all our boxes out!

Yes there are far more geo-caches in NZ - last time I looked there was over
300 but there is probably many more now. You'll find that many geo-caches
can be done without a GPS :-D
I don't think it's a problem to use locations where there are geo-caches,
but you want to make sure your box is well away from the geo-cache and that
it is clearly labelled that it's a letterbox (there is a flyer in the yahoo
files section you could print out and include in the box). There have been
many occassions in the states where geo-cachers have found a nearby
letterbox and have taken the stamp thinking that it was treasure. :-(

Cheers,
